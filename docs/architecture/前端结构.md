# 前端结构

> 截至 v1.0.0+。与 `项目结构.md` 配合看。

## 依赖方向

```
main.ts ─→ App.vue ─→ views/ ─→ api/  ─→ @tauri-apps/api/core
                       ├─→ stores/
                       ├─→ components/
                       └─→ router/

views/  ─→ components/  (复用 UI 块)
api/    ─→ stores/      (设置缓存在 store)
stores/ ─→ api/         (读 / 写后端)
```

单向、无环。**`views/` 不直接调 `api/` 之外的 IO**（`mock/` 在 invoke 失败时回退，是 api/core 内部的事）。

**Init 守卫**：`main.ts` 启动时调 `tauri_check_init_default()`；如果未初始化，`router.push('/init')` 强制走 InitView，初始化完跳 `/review`。
**Dev 路由**：`Sidebar` 与 `App.vue` 根据 `settingsStore.developerMode`（运行时，**不持久化**）决定是否显示。

## 视图（`src/views/`）逐个说明

| View | 职责 | 关键交互 |
|---|---|---|
| `InitView` | 首次选择数据目录 / 调用 `tauri_init_note` | Tauri 启动流程入口 |
| `ReviewView` | 今日推荐入口：拉 `recommend_questions_comm` | 跳 `/review/session` |
| `ReviewSessionView` | 单题答题：CORRECT/WRONG/FUZZY → `process_review_comm` | 完成跳 `/review/summary` |
| `ReviewSummaryView` | 本次复习统计：CORRECT/WRONG/FUZZY 计数 | 点题跳 `/questions/:id` |
| `QuestionsView` | 题目列表：分页 8/页，URL 同步，搜索 / 科目 / 状态 / 高频错题筛选，**自带过滤+分页** | 行跳详情 |
| `QuestionDetailView` | 题目详情：图片、编辑、删除、suspend/recover | 各种编辑操作都走 `goBack` 返回 |
| `QuestionNewView` | 新建题目：选图、上传 | `handleCancel` 走 `goBack` |
| `RecycleBinView` | 回收站：恢复、永久删除 | 列表跳详情 |
| `RecycleBinDetailView` | 回收站详情：恢复、永久删除 | `goBack` 返回 |
| `SettingsView` | 设置：复习参数 / 科目池 / 时区 / 切日 | 高级设置折叠（debug / 推荐随机性 / 显示调试 / 时区 / 切日） |
| `StatsView` | 统计：4 个概览卡 + 题目状态分布 + 分科复习状况 + 月度折线图 | 折线图按月切换（←/→/回到本月） |
| `DevCenterView` | 开发者中心：进入推荐调试 | dev only |
| `DevRecommendationListView` | 今日推荐列表（dev 视图） | dev only |
| `DevPreviewRecommendationView` | 推荐预览：所有题评分 / 入选 / 落选 reason | dev only |

## 通用工具（`src/utils/back.ts`）

```ts
import type { Router } from 'vue-router'

export function goBack(router: Router, fallback: string): void {
  if (window.history.state?.back) {
    router.back()
  } else {
    router.push(fallback)
  }
}
```

所有"返回"按钮统一调 `goBack(router, fallback)`，避免直接 `router.push` 硬路径丢失栈。

## 主题样式

集中在 `assets/main.css` + 各 view 的 `<style scoped>`。`Sidebar` 固定 200px 宽，主区 `margin-left: 200px`。

主色：
- 绿色 `#4CAF50`（主操作 / 推荐通过）
- 蓝色 `#2196F3`（题目数 / 信息）
- 橙色 `#FF9800`（复习次数 / 警告）
- 紫色 `#9C27B0`（准确率 / 数据）

## Mock 回退（`src/api/core.ts`）

```ts
export async function call<T>(cmd: string, payload?: any): Promise<T> {
  if (isTauri()) {
    return await invoke<T>(cmd, payload)
  }
  return await mockFallback(cmd, payload)  // 走 docs/data.ts
}
```

`isTauri()` 检测 `@tauri-apps/api/core::isTauri()`。Mock 模式下前端仍可运行，便于 dev 调试 UI（业务数据是假的）。
