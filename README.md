# ePNote 电子错题本

一个高效的错题管理与复习系统，帮助您系统性地复习和掌握重点知识。

## 功能特性

### 题目管理

- 创建、编辑、删除题目
- 支持添加题目图片和答案图片
- 支持多知识点管理
- 支持按科目、状态筛选题目

### 复习系统

- 智能复习推荐
- 题目状态跟踪（新题/学习中/已掌握/待复习/暂停）（暂停功能暂未实现）
- 复习结果记录
- 学习进度统计

### 回收站

- 误删恢复功能
- 永久删除功能
- 30天自动清理（可选）

## 技术栈

- **前端**: Vue 3 + TypeScript + Vite
- **后端**: Rust + Tauri 2.x
- **数据库**: SQLite
- **状态管理**: Pinia
- **路由**: Vue Router

## 项目结构

```
ePNote/
├── src/                    # 前端源代码
│   ├── api/               # API 调用
│   ├── components/        # 公共组件

│   ├── types/             # TypeScript 类型定义
│   └── views/             # 页面视图

│   │   ├── dao/          # 数据访问层

```

## 开发环境要求

- Node.js >= 20.19.0
- Rust >= 1.70
- Windows 10/11 (当前支持)

## 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 开发模式

```bash
# 前端开发
npm run dev

# Tauri 开发（同时启动前后端）
npm run tauri dev
```

### 3. 构建生产版本

```bash
npm run tauri build
```

## 使用说明

### 首次使用

1. 启动应用后，选择数据存储目录
2. 点击"初始化系统"
3. 重启应用程序

### 主要功能入口

- **复习**: 主页点击"开始复习"进入复习会话
- **题目管理**: 点击顶部导航栏进入题目列表
- **新建题目**: 在题目管理页面点击"新建题目"
- **回收站**: 在题目管理页面点击"回收站"

### 题目状态说明

| 状态 | 说明 |
|------|------|
| 新题 | 初次添加，还未复习 |
| 学习中 | 正在复习掌握中 |
| 已掌握 | 连续答对多次 |
| 待复习 | 间隔一段时间需要复习 |
| 暂停 | 暂时跳过不复习 |

## 配置说明

配置文件位于 `src-tauri/app_config.json`:

```json
{
  "root": "数据存储路径"
}
```

## 常见问题

### Q: 初始化后无法使用？

A: 初始化成功后需要重启应用程序才能使配置生效。

### Q: 如何备份数据？

A: 复制整个数据存储目录即可完成备份。

### Q: 回收站会自动清理吗？

A: 可以调用清理接口，默认清理30天前的永久删除题目。
